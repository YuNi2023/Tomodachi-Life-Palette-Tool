使い方

01 / 画像を読み込む

枠に画像をドロップ、または枠をタップしてファイルを選択。手元に画像が無ければ デモ画像で試す から開始できます。

02 / 1:1 に切り出す（正方形でない場合）

  • 角の □ ハンドルでサイズ変更（縦横比は1:1で固定）
  • 枠の内側をドラッグで位置調整
  • この範囲で確定：選択範囲を切り出してメイン画面へ
  • 切り出さずに使う：元画像のままメイン画面へ
  • 中央リセット：選択枠を最大の正方形に戻す
  • 切り出し後、メイン画面の クロップやり直し でいつでも再選択可能

03 / 原画 / 変換 を切り替える

  • 原画：アップロード／クロップ済みの画像をそのまま表示
  • ゲームパレット変換：画像をゲーム内84色で塗り直したプレビュー（グリッドサイズ16〜64選択可、ディザのON/OFF切替可）

04 / ピクセルの色を取得する

  1. 選択モードのまま画像内のピクセルをタップ／クリック
  2. 右側に HEX / RGB / HSV と最も近いパレット色が表示

05 / 使用色レシピ・ハイライト

変換ビュー時、下部に使用色レシピが表示。レシピ行 or パレットセルをhover/タップで画像内の該当セルが浮かび上がります。

06 / HEX直接入力

右上の HEX 欄にコードを貼り付けて「確認」を押せば、画像クリックなしでも HSV 手順を表示。#fab のような短縮形にも対応。

07 / パレット色 / フルカラー切替

  • カラーパレット：ゲーム内84色のうち最も近い色を表示
  • フルカラー：色相スライダーと彩度×明度の操作回数の目安を表示

08 / 表示の補助機能

  • 移動モード：拡大表示時にドラッグでスクロール
  • ルーラー：画像中心線とX/Y座標番号を表示
  • ＋ / −：表示倍率（1× / 2× / 4× / 8× / 12× / 16×）
