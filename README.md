# すぽいと帳

🌍 日本語 / English / 한국어 / Français / Español / 繁體中文 / 简体中文 / Deutsch / Italiano / Nederlands / Português (BR) / Русский

トモダチコレクション わくわく生活のペイント機能で使える、画像 → ドット絵 変換ツール。

任意の画像をゲーム内 84 色パレットに近似変換し、HSV 操作手順や色番号塗り絵まで生成します。

## 主な機能

### 入力

- ドラッグ&ドロップ または クリックでアップロード（PNG / JPG / GIF / WEBP）
- 用途に合わせた **5 種のアスペクト比** からクロップを選択:
  - **1:1**（256×256）— 通常のプロフィール画像
  - **本** 180×256 — 縦長ブック
  - **TV** 256×131 — テレビ画面
  - **ゲーム** 256×144 — ゲーム画面風
  - **壁紙** 172×256 — 縦長壁紙
- ハンドルでサイズ変更、内側ドラッグで位置調整
- いつでも「クロップやり直し」で再クロップ

### 変換

- ゲーム内 84 色パレットへの自動変換（ディザ ON/OFF 切替）
- 2 系統のブラシサイズ:
  - **ゲーム準拠**（Pixel-perfect）— 4px / 8px / 16px / 32px ・ ゲーム内ブラシと一致
  - **中間サイズ**（Smooth）— 1px / 3px / 7px / 13px / 19px / 27px ・ より細かい刻み

### 表示

- ピクセル単位のクリック / タップで色取得（HEX / RGB / HSV）
- ゲーム内 HSV 操作手順の自動算出
- 使用色レシピ（変換ビュー時、各色のセル数）
- パレット ↔ レシピの hover ハイライト連動
- HEX コード直接入力で色をパレット上に位置表示
- **グリッド表示** — 1 セル / 8 セル区切り / 中央線 の階層的オーバーレイ
- **外部ルーラー** — 行 / 列の位置番号を欄外表示、ズーム・拡張比率に応じて目盛間隔を自動調整

### 出力

- **ドット絵 PNG** — ロゴ・タイトル帯付き、SNS シェア向け
- **番号塗り絵 PNG**（Paint-by-Number）— 各セルに色番号 + 凡例付き、印刷して手塗り可能

### 多言語

12 言語対応、ブラウザ言語による自動判定 + 手動切替可能。

## 多言語対応

12 言語に対応しています。ヘッダー右上の歯車アイコンから切替できます。

| | |
|---|---|
| 日本語 | English |
| 한국어 | Français |
| Español | 繁體中文 |
| 简体中文 | Deutsch |
| Italiano | Nederlands |
| Português (BR) | Русский |

初回アクセス時はブラウザの言語設定から自動判定されます。選択した言語はブラウザに保存され、次回以降も維持されます。

**翻訳 PR 歓迎です。** 翻訳の改善や新しい言語の追加方法は [docs/I18N.md](./docs/I18N.md) を参照してください。

## 技術

すべてブラウザ内で動作。サーバー処理なし、ユーザーの画像はどこにも送信されません。

- Vanilla JavaScript（フレームワーク不使用）
- HTML / CSS / JS のみ、ビルドツール不要
- 静的ホスティング前提（GitHub Pages など）

設計判断と「何をやらなかったか」の詳細は [docs/DESIGN.md](./docs/DESIGN.md) を参照してください。

オンラインで使う: https://yu08083.github.io/Tomodachi-Life-Palette-Tool/
