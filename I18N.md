# 多言語対応（i18n）

すぽいと帳は **12言語** に対応しています。

| コード | 言語 | ネイティブ表記 |
|---|---|---|
| `ja` | 日本語 | 日本語 |
| `en` | English | English |
| `ko` | 韓国語 | 한국어 |
| `fr` | フランス語 | Français |
| `es` | スペイン語 | Español |
| `zh-TW` | 繁体中文 | 繁體中文 |
| `zh-CN` | 簡体中文 | 简体中文 |
| `de` | ドイツ語 | Deutsch |
| `it` | イタリア語 | Italiano |
| `nl` | オランダ語 | Nederlands |
| `pt-BR` | ブラジルポルトガル語 | Português (BR) |
| `ru` | ロシア語 | Русский |

## 言語の決まり方

1. ユーザーが画面右上の歯車アイコン → ドロップダウンで明示的に選んだ言語（LocalStorageに保存）が最優先
2. 上記がない場合、ブラウザの言語設定（`navigator.languages`）から自動判定
3. それでも該当がなければ日本語にフォールバック

### 地域コード対応の特殊ロジック

- `zh-Hant` `zh-HK` `zh-MO` → `zh-TW`（繁体字圏として一括）
- `zh-Hans` `zh-CN` `zh-SG` `zh-MY` → `zh-CN`（簡体字圏として一括）
- `pt-BR` → `pt-BR`（ポルトガル本国のptはまだ未対応）
- `en-US` `en-GB` `en-AU` 等 → `en`
- 同様に他の言語コード+地域も基本コードにフォールバック

## ファイル構成

```
│   └── i18n.js                  ← 翻訳エンジン
└── i18n/

    ├── en.json

    ├── zh-TW.json               ← 繁体中文

```

## UIの仕組み

ヘッダー右上の **歯車アイコン + 現在言語名** をクリックするとドロップダウンメニューが開き、対応言語の一覧から選択できます。選択中の言語にはチェックマーク（✓）が表示されます。

「国旗で言語を表現しない」原則に従い、テキストとアイコンのみで構成しています。

## 翻訳の仕組み

HTML側に `data-i18n="key"` 属性を付けるだけで、起動時に対応する言語の文字列に置換されます。

```html
<button data-i18n="upload.demo">デモ画像で試す</button>
```

属性ベースの翻訳パターン:

| 属性 | 用途 |
|---|---|
| `data-i18n="key"` | textContentを置換 |
| `data-i18n-html="key"` | innerHTMLを置換（`<br>`等を含む文用） |
| `data-i18n-attr-aria-label="key"` | aria-label属性を翻訳 |
| `data-i18n-attr-placeholder="key"` | placeholder属性を翻訳 |
| `data-i18n-attr-title="key"` | title属性を翻訳 |

JS側からの呼び出し:

```javascript
// シンプル
alert(t('crop.loadFail'));

// プレースホルダ補間
posEl.textContent = t('color.rowCol', { row: 5, col: 12 });
// → "5行目 12列目" / "row 5, col 12" / "5행 12열" / "ligne 5, col. 12" など
```

## 新しい言語を追加する

1. `js/i18n/ja.json` をコピーして `xx.json` を作成（xxは [ISO 639-1コード](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes)、地域つきなら `xx-YY` 形式）
2. すべての値を翻訳
3. `js/core/i18n.js` の `I18N_SUPPORTED` 配列と `I18N_NATIVE_NAMES` オブジェクトに追加
4. （オプション）地域マッピングが必要なら `i18nDetect()` 内の `REGION_MAP` に追加
5. PRを送る

ドロップダウンメニューは `I18N_SUPPORTED` 配列から自動生成されるので、HTMLの編集は不要です。

### 翻訳時の注意

- **キー構造はja.jsonと完全一致** させてください（不足キーがあるとUIが壊れます）
- `{name}` のような波括弧プレースホルダは**そのまま残す**こと
- HTMLタグ（`<strong>` `<br>` `<a>` 等）は意味を保ったまま位置調整可
- ゲーム名は各国の正式名称を使用（例: 仏語/英語/独語/西語/伊語/蘭語/葡語は「Tomodachi Life」、韓国・繁体中文は未公式販売のため意訳）
- ブランディング部分（PNG出力に焼き込まれる「すぽいと帳」「トモダチコレクション...」）は**翻訳されません**。これはツールのアイデンティティとして日本語固定の意図です

## 既知の制約

- ブラウザの右クリック「翻訳」機能を使うと二重翻訳になります（言語スイッチャーをご利用ください）
- 言語変更時、既に保存した「お気に入り色」のHEXコードは保持されますが、ラベルは新言語で再描画されます
- `<meta name="description">` は静的なため、検索エンジン向けには日本語のみ。SEO拡張は今後の課題

## 翻訳品質について

韓国語・フランス語・スペイン語・繁体中文・ドイツ語・イタリア語・オランダ語・ブラジルポルトガル語の各翻訳は、文意ベースの叩き台です。各言語のネイティブスピーカーによる改善PRを歓迎しています。

特に以下のような点は要改善ポイントの可能性があります:
- 専門用語（「ディザ」「彩度」「明度」など）の慣用表現
- ゲーム機コントローラの記号（ZL/ZR）の各国版表記
- 句読点・スペースの慣習（フランス語のコロン前スペース等）

PRやIssueでお知らせください。
